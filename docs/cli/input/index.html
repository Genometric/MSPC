Input

MSPC takes as input a tab-delimited file in BED format for each replicate, each containing enriched regions (aka peaks) called with a permissive p-value threshold.

By default, the columns of the BED files should comply the following field order:

IndexName
0chrom
1chromStart
2chromEnd
3name
4value

The p-value needs to be in -Log10(p-value) format. More columns can be present, but they will not be parsed.
